B站鸣潮视频数据采集指南
=========================

由于B站的反爬虫机制，直接爬取可能被限制。请按照以下步骤操作：

## 方法一：自动脚本（推荐先尝试）

1. 运行脚本：
   ```bash
   python bilibili_video_scraper.py
   ```

2. 脚本会尝试通过API获取数据，如果成功会自动下载封面并生成CSV文件

3. 如果API被限制，会自动切换到手动输入模式

## 方法二：手动操作（更可靠）

### 步骤1：浏览器访问
1. 打开Edge浏览器
2. 访问：https://www.bilibili.com
3. 在搜索框输入"鸣潮"

### 步骤2：收集数据
在搜索结果页面，按以下方式收集TOP 10视频：

**需要记录的信息：**
- 视频标题
- 作者名称
- 播放量
- 时长
- BV号（如：BV1xx411c7mD）
- 封面图片URL（右键封面→复制图片地址）
- 视频链接（可从BV号自动生成）

### 步骤3：运行脚本输入数据

```bash
python bilibili_video_scraper.py
```

按照提示依次输入收集到的视频信息。

### 步骤4：查看结果

脚本会生成：
- `mingchao_videos_YYYYMMDD_HHMMSS.csv` - 视频数据表格
- `covers/` 目录 - 封面图片文件

## 方法三：使用开发者工具（技术方案）

如果需要更精确的数据：

1. 按F12打开开发者工具
2. 切换到Network标签
3. 在B站搜索"鸣潮"
4. 找到API请求（通常是 `api.bilibili.com/x/web-interface/search/type`）
5. 复制请求URL和Headers
6. 修改脚本使用这些请求参数

## 文件格式说明

生成的CSV文件包含以下列：
- 标题：视频标题
- 作者：UP主名称
- 播放量：观看次数
- 时长：视频时长
- BV号：B站视频唯一标识
- 链接：视频完整URL
- 封面URL：封面图片地址
- 上传时间：发布日期
- 弹幕数：弹幕总数
- 点赞数：点赞数量

## 注意事项

1. **反爬虫问题**：B站可能会限制API请求频率，如果失败请使用手动模式
2. **封面下载**：部分封面可能无法直接下载，需要手动保存
3. **数据时效性**：热门视频会动态变化，建议定期更新数据
4. **合规使用**：请遵守B站的使用条款，仅用于个人学习研究

## 替代方案

如果Python脚本无法满足需求，可以考虑：

1. **Excel导入**：
   - 手动从浏览器复制数据到Excel
   - 使用Excel的"从Web获取数据"功能

2. **浏览器扩展**：
   - 使用"B站视频信息导出"类浏览器扩展
   - 可以批量导出视频信息

3. **第三方工具**：
   - 如youtube-dl（支持B站）
   - bilibili-api-rs等开源项目

## 快速开始

```bash
pip install requests

```
