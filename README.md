## 获取封面图片的方法

对于每个视频,可以通过以下方式获取封面图:

1. **从B站链接提取**: B站视频的封面图通常有固定格式
2. **访问视频页面**: 打开视频链接,右键保存封面图片
3. **使用截图工具**: 在视频播放界面截图封面

## 示例封面图链接格式

B站视频封面图格式示例:
```
https://i2.hdslb.com/bfs/archive/[视频ID]_[尺寸].jpg
```

例如: `https://i0.hdslb.com/bfs/archive/8c8f8c8f8c8f8c8f8c8f8c8f8c8f8c8f8c8f8c8f.jpg`
